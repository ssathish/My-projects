*PHP web application for learning disabled children.
*This is not a fully completed application and is in developing stage.

Desciption:

*This is a web based application for teachers to co-ordinate with learning disabled children.
*This application uses Google id for login using OAuth. 
*Once logged in the google calendar of the user is used to add events and mark events if completed.
*A separate MySQL database is also used by PHP to keep track of the students,teachers and the event details and the progress of students can be tracked by teacher.
